# .kiro 目录说明

这是Kiro IDE的配置和文档目录。

## 📁 目录结构

```
.kiro/
├── docs/              # 项目文档
│   ├── README.md      # 文档索引
│   ├── DEVELOPMENT.md # 开发环境配置指南
│   └── DOCKER_COMPARISON.md # Docker运行方式对比
│
├── specs/             # 功能规格说明
│   └── documentation-completion/ # 文档完善规格
│
└── README.md          # 本文件
```

## 📚 快速导航

- [Docker对比说明](./docs/DOCKER_COMPARISON.md) - 生产vs开发环境

## 🔧 Kiro IDE配置

本目录也用于存放Kiro IDE的相关配置：
- Specs（规格文档）
- Steering（引导规则）
- Skills（技能定义）
- Hooks（钩子配置）

## 📝 文档规范

- 使用Markdown格式
- 中英文混排时注意空格
- 代码块指定语言类型
- 保持文档与代码同步
