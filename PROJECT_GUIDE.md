# 🎓 KnowledgeMap AI - 项目说明文档

> **让 AI 为你构建专属的知识宇宙**
> *一个将关键词转化为系统化知识图谱的 AI 个性化学习平台。*

---

## 🌟 1. 产品概述

**KnowledgeMap AI** 是一个基于生成式 AI 的深度化学习平台。与传统的聊天机器人不同，它不满足于碎片化的问答，而是致力于将用户的**一个关键词**（如“量子计算”、“西方哲学史”）转化为一套**系统化、结构化、可视化**的完整课程体系。

**核心价值**：解决碎片化知识难以成体系的痛点，为终身学习者提供从“入门”到“精通”的一站式学习解决方案。

---

## 🔄 2. 核心交互流程

我们的交互设计遵循“**由薄到厚，由点到面**”的认知规律，引导用户逐步深入。

### 2.1 第一步：极简入口与意图识别 🎯
- **交互动作**：用户在首页输入框输入任意感兴趣的主题关键词（例如“深度学习”）。
- **个性化配置**：
  - **难度选择**：提供 **入门** / **进阶** / **专家** 三种模式，AI 智能调整深度。
  - **学习目标**：用户可简述具体需求（如“我想在一个月内掌握基础”），定制化大纲重点。
- **系统反馈**：点击“生成课程”后，系统立即响应，无缝进入加载状态。

### 2.2 第二步：动态大纲生成 🌳
- **视觉呈现**：屏幕左侧的 **课程大纲树** 实时生长。
- **流式体验**：采用 Stream 流式传输，章节节点一个个“蹦”出来，缓解等待焦虑，过程极具观赏性。
- **结构化展示**：自动生成 **章** 与 **节** 的嵌套结构，支持自由折叠/展开。

### 2.3 第三步：沉浸式内容学习 📖
- **交互动作**：点击左侧大纲中的任意节点。
- **智能生成**：
  - **秒级响应**：已有内容从本地缓存瞬间加载。
  - **实时创作**：新内容由 AI 引擎实时生成，配合**打字机动画**，模拟老师板书，引导视线。
- **富文本体验**：
  - **公式完美渲染**：支持复杂的 LaTeX 数学公式（自动渲染矩阵、积分等）。
  - **可视化图表**：AI 自动插入 Mermaid 流程图、时序图，将抽象概念可视化。
  - **代码高亮**：支持多种编程语言的语法高亮与一键复制。

### 2.4 第四步：知识图谱导航 🕸️
- **交互动作**：点击知识图谱生成按钮，触发知识图谱构建。
- **全景视野**：点击右上角切换视图，系统构建出动态的 **网络拓扑图**。
- **实体与关系**：直观展示核心概念、定理、人物及其相互连接（如 A 是 B 的前置知识）。
- **双向联动**：点击图谱节点可直接跳转至对应课程章节，支持非线性的探索式学习。

### 2.5 第五步：智能助教与测验 🤖
- **侧边栏互动**：右侧 **助教面板** 随时待命。
- **上下文感知**：助教自动读取当前章节内容，用户直接提问“这句话什么意思？”，AI 即可精准作答。
- **角色扮演**：支持设置助教画像（如“苏格拉底”、“严厉教授”），定制交互风格。

## ✨ 3. 交互设计亮点

### 3.1 响应式与自适应布局 📱
- **三栏式 IDE 布局**：左大纲 + 中正文 + 右助教，专为沉浸式学习打造。
- **自由定制**：左右侧边栏宽度支持拖拽调整，满足不同阅读习惯。
- **移动端适配**：手机端自动折叠侧边栏，通过手势和菜单呼出，保证核心体验不打折。

### 3.2 视觉反馈与微交互 🎨
- **状态感知**：细致的 Loading 骨架屏、进度条，让等待不再枯燥。
- **极客状态栏**：底部实时显示 Token 消耗、生成速度，增加科技感与透明度。
- **无感保存**：学习进度、对话记录、笔记自动持久化，刷新页面零丢失。

### 3.3 深度定制的 Markdown 渲染 📝
我们针对学术场景进行了深度优化：
- **公式防抖**：彻底解决 LaTeX 公式渲染时的闪烁问题。
- **图表交互**：Mermaid 图表支持缩放、拖拽，查看大图更清晰。
- **代码增强**：增加一键复制、语言标识等实用功能。

### 3.4 全自动学习足迹记录 👣
**（🔥核心亮点）** 系统自动记录每一次思维跳跃，无需手动整理：
- **✍️ 自动生成笔记**：AI 自动识别核心知识点生成精炼笔记卡片，对话内容也可一键转笔记。
- **❌ 智能错题本**：随堂测验错题自动归档，附带 AI 深度解析与知识点回溯，复习更高效。
- **📊 自动总结功能**：章节学习结束自动生成“学习简报”（思维导图+核心概念+掌握度评估）。
- **📈 数据可视化**：多维度图表展示学习时长、覆盖率与成长曲线，让进步看得见。

## 🛠️ 4. 技术架构简述

为了实现流畅体验，我们采用了现代化的技术栈：

- **前端**：
  - **Vue 3 + TypeScript**：坚实的工程化基础。
  - **Pinia**：复杂状态管理（课程树、聊天记录、设置）。
  - **Stream Parsing**：自研流式解析器，实现 SSE 到 UI 的平滑更新。
  
- **后端**：
  - **FastAPI**：高性能异步 Python 框架。
  - **Prompt Engineering**：精心设计的 PromptTemplate 系统，确保 AI 输出格式（JSON/Markdown）严格可控。
  - **Task Manager**：异步任务队列，保障长内容生成不阻塞。

## 📝 5. 评测建议

**KnowledgeMap AI** 试图还原人类认知的自然过程：从一个关键词出发，构建起宏大的知识大厦。

建议您重点体验以下流程：
1.  🏁 **生成一份新课程**：观察大纲生成的逻辑性与流式效果。
2.  📖 **点开一个章节**：体验打字机生成、公式渲染与图表展示。
3.  🕸️ **打开知识图谱**：感受知识点之间的关联与跳转。
4.  👣 **查看“我的笔记”**：体验全自动记录的学习足迹与错题本功能。
5.  💬 **与助教对话**：测试其对当前课程内容的理解与上下文感知能力。
